# 功能概览

## 仪表盘 (Dashboard)
仪表盘提供了系统的整体状态概览，包括服务健康状态、快捷入口和系统信息。

![Dashboard](/screenshots/home.png)

## 插件市场 (Market)
在插件市场中，您可以浏览、安装和管理各种插件，扩展 CCR 的功能。

![Market](/screenshots/market.png)

## MCP 服务器 (MCP Servers)
管理您的 Model Context Protocol (MCP) 服务器配置，查看连接状态和可用工具。

![MCP Servers](/screenshots/mcp.png)

## 代理与客户端 (Agents & Clients)
配置和管理 AI 代理及客户端连接，包括 Claude、Codex 等。

![Agents](/screenshots/agents.png)

## 设置 (Settings)
全量配置 CCR 的各项参数，包括通用设置、网络配置和高级选项。

![Settings](/screenshots/settings.png)
